## Current Babagigi wave assignments

**Wave 1 (Commercial Launch):** Spanish, Mandarin, Tagalog, Vietnamese, Korean, Arabic, French, Portuguese

**Wave 2 (Fast Follow):** Cantonese (deserves separate treatment from Mandarin), Russian, Polish, Italian, Hindi

**Wave 3 (Depth):** German, Japanese, Greek, Hebrew, Punjabi, Urdu, Gujarati

**Wave 4 (Mission — cross-subsidized):** Khmer, K'iche', Lao, Hmong, Nahuatl, Mixtec

## Key decisions

- **Tagalog elevated to Wave 1** — 33% of speakers are 60+, strongest grandparent demographic
- **Cantonese separated from Mandarin** — distinct communities, distinct demographics, distinct preservation urgency
- **European heritage languages (German/Italian/Polish) moved down** despite high speaker counts because transmission is past peak
- **Wave 4 mission-track languages enabled by Omnilingual** — November 2025 release moved Khmer, K'iche', Hmong from "experimental/none" STT to "usable"

## Status

- Wave 1: In development (Q2 2027 target launch)
- Wave 2: Planning
- Wave 3-4: Architectural support exists; specific timeline TBD

## How to add a new language

A formal protocol exists in migration 006: `OPP-NEW-LANGUAGE` (Should We Add a New Language?). It requires evaluating:
- Tech readiness state
- Partnership commitment
- Existing capacity displacement
- Community voice in the decision
